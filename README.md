# Description

BookManager is a web application that provides a simple solution for managing books with comfortable admin panel. It allows users to keep track of their book collections, add new books, update book details, and remove books from the collection, and also sell books.

The application is built with CodeIgniter 4, a powerful PHP framework known for its speed, reliability, and ease of use.

Whether you're a book lover with a vast collection or a small library looking to digitize your catalog, BookManager is the simple and efficient tool for you.
